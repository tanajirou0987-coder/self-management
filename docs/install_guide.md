# アプリのインストール方法（PWA）

このアプリはPWA（Progressive Web App）として動作し、スマホのホーム画面にインストールできます。

## 📱 Android（Chrome）でのインストール

### 手順

1. **スマホのChromeブラウザでアプリにアクセス**
   - VercelのURL（例: `https://self-management-app.vercel.app`）を開く

2. **ログイン**
   - ログイン画面でパスコードを入力してログイン

3. **インストールプロンプトを確認**
   - 画面下部に「アプリをインストール」というプロンプトが表示されます
   - 表示されない場合は、次の手順4を試してください

4. **手動でインストール**
   - ブラウザメニュー（右上の3点アイコン）をタップ
   - 「アプリをインストール」または「ホーム画面に追加」を選択

5. **確認ダイアログ**
   - 「インストール」をタップして確認

6. **完了**
   - ホーム画面にアプリのアイコンが追加されます
   - アイコンをタップすると、ブラウザではなくアプリとして起動します

### スクリーンショットの例

```
[Chromeブラウザ] → [メニュー] → [アプリをインストール] → [インストール]
```

## 🍎 iOS（Safari）でのインストール

### 手順

1. **スマホのSafariブラウザでアプリにアクセス**
   - VercelのURL（例: `https://self-management-app.vercel.app`）を開く

2. **ログイン**
   - ログイン画面でパスコードを入力してログイン

3. **共有ボタンをタップ**
   - 画面下部の共有ボタン（□↑アイコン）をタップ

4. **「ホーム画面に追加」を選択**
   - メニューの中から「ホーム画面に追加」を探してタップ

5. **名前を確認**
   - アプリの名前が表示されます（必要に応じて変更可能）
   - 「追加」をタップ

6. **完了**
   - ホーム画面にアプリのアイコンが追加されます
   - アイコンをタップすると、Safariではなくアプリとして起動します

### スクリーンショットの例

```
[Safariブラウザ] → [共有ボタン □↑] → [ホーム画面に追加] → [追加]
```

## ✅ インストール後の使い方

- **ホーム画面のアイコンから起動**: ブラウザではなくアプリとして動作します
- **オフライン対応**: Service Workerにより、オフラインでも基本的な機能が利用できます
- **フルスクリーン表示**: ブラウザのアドレスバーが表示されず、アプリのような体験ができます

## 🔐 ログインについて

- **初回アクセス時**: ログイン画面でパスコードを入力する必要があります
- **ログイン状態の維持**: ログイン後、30日間は自動的にログイン状態が維持されます
- **ログアウト後**: ログアウトすると、次回アクセス時に再度ログインが必要です

## ❓ インストールできない場合

### 確認事項

1. **HTTPS接続**: アプリはHTTPSでアクセスする必要があります（Vercelは自動的にHTTPS）
2. **ブラウザ対応**: Chrome（Android）またはSafari（iOS）を使用してください
3. **キャッシュ**: ブラウザのキャッシュをクリアして再試行してください

### トラブルシューティング

- **プロンプトが表示されない**: 
  - ブラウザメニューから手動でインストールを試してください
  - ページをリロードしてから再度試してください

- **インストールボタンが見つからない**:
  - Android: ブラウザメニュー → 「アプリをインストール」
  - iOS: 共有ボタン → 「ホーム画面に追加」

- **エラーが表示される**:
  - インターネット接続を確認してください
  - ブラウザを最新版に更新してください

## 📝 注意事項

- PWAはブラウザベースのアプリです。App StoreやGoogle Playからはダウンロードできません
- インストール後も、定期的にインターネット接続が必要です（データ同期のため）
- データはブラウザのローカルストレージとSupabaseに保存されます
